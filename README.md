# Daily Mood Tracker

A simple web app where users select a mood, write a short note, and save it in the browser. Each entry is stored using **localStorage**, so the data stays even after refreshing the page.

## Features
- Choose a mood (Happy, Okay, Sad, Stressed, Excited)
- Add a short note
- Save entries with localStorage
- View saved mood history
- Clear all entries

## Technologies Used
- HTML
- CSS
- JavaScript (DOM, events, localStorage)
